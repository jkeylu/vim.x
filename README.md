vim.x
=====

### Install

1. Setup Vim.x

    ```
    $ git clone https://github.com/jkeylu/vim.x.git ~/.vim
    $ cd ~/.vim
    $ ./install.sh
    ```
2. Install configured bundles

    Launch `vim`, run `:PlugInstall`
    (or `vim +PlugInstall +qall` for CLI lovers)

3. Update bundles

    Lunch `vim`, run `:PlugUpdate`
