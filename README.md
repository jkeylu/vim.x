vim.x
=====

### Install

1. Setup Vim.x

    ```
    $ git clone https://github.com/jkeylu/vim.x.git ~/.vim
    $ cd ~/.vim
    $ ./vimx
    ```
2. Install configured bundles

    Launch `vim`, run `:BundleInstall` 
    (or `vim +BundleInstall +qall` for CLI lovers)

3. Update bundles

    Lunch `vim`, run `:BundleInstall!`
